Commodity Trader

How We Stopped a Commodity Trader’s Undetected Fuel Leakage

38 vehicles across multiple subcontractors. No reliable fuel monitoring. Losses accumulated undetected until we built a live exception-driven dashboard.

38+Vehicles
R200k+Waste identified
WksTo first detection

What was built

A Python-based fuel monitoring API connected to a real-time web dashboard. Fuel issued vs consumed, distance vs usage, time-based trip analysis, and automated exception alerts — all running without manual input.

Outcomes

  • Fuel leakage detected within weeks of go-live
  • Full cost visibility across every vehicle & partner
  • Partners held accountable against objective data
